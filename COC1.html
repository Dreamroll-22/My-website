COC1: Install and Configure Computer Systems and Hardware

Computer Assembly and Disassembly

Computer Disassembly

  1. Remove the Power Cable – Make sure the computer is turned off before disassembling.
  2. Remove the Cover – Open the case using the right tools.
  3. Unplug Cables and Connectors – Remove all internal cables carefully.
  4. Take Out Adapters and Drives – Unscrew and unplug expansion cards and storage devices.
  5. Take Out the Memory Modules (RAM) – Unclip and take out the memory sticks.
  6. Disconnect the Power Supply – Unscrew and remove from the motherboard.
  7. Remove the Motherboard – Remove the screws and lift it out carefully.

Computer Assembly

  1. Get Ready for Assembly
    • Count all parts.
    • Make sure to have a clean, static-free environment.
    • Have tools and drivers at hand.
  2. Get the Motherboard Ready
    • Unbox and check for damage.
    • Put it on an anti-static surface.
  3. Install the CPU
    • Align and fix the processor in its socket.
  4. Install CPU Heat Sink
    • Apply the thermal paste and affix the cooler.
  5. Install Memory (RAM Modules)
    • Put RAM sticks in their respective slots.
  6. Install the Motherboard into the Case
    • Align over standoffs and fasten using screws.

Installation of Operating System and Software

Creating a Bootable USB

  1. Plug the USB Flash Drive into the computer.
  2. Open "This PC", right-click the USB drive, and click "Format".
  3. Launch Rufus.
  4. Click the CD icon in the lower right corner and select your ISO file.
  5. Click "Start" and confirm by clicking "Yes".
  6. Wait for the process to complete. A "Ready" status will appear once done.

Installation of Windows 10

  1. Turn on the computer and insert the bootable USB.
  2. Enter BIOS and change the boot order.
  3. Press any key to boot from USB.
  4. Click "Install Now".
  5. Select language and input product key (or skip).
  6. Accept the license agreement.
  7. Select "Custom: Install Windows only (advanced)".
  8. Partition the hard drive and select a drive for installation.
  9. Wait for installation to complete and let the system restart.
  10. Re-enter BIOS and set HDD as the first boot device.
  11. Follow personalization settings and create a user account.
  12. Complete setup and Windows 10 will launch successfully.

Installation of Windows Server 2012 R2

  1. Turn on the computer.
  2. Insert a bootable USB drive.
  3. Enter BIOS and change the boot order.
  4. Select the USB drive as the first boot device.
  5. Save changes and restart.
  6. Boot from the USB and begin installation.
  7. Choose installation language and click "Install Now."
  8. Select Windows Server 2012 R2 Standard (Full Installation).
  9. Accept the licensing agreement.
  10. Configure partitions and select the target drive.
  11. Begin installation and wait for the process to complete.
  12. Restart and set the boot priority back to the hard drive.
  13. Complete the initial setup, including password configuration.
  14. Windows Server 2012 R2 is now successfully installed.

Installation of Application Software

  1. Prepare the Software Installer
    • Visit the software's official website.
    • Click the Windows download link.
    • Wait for the file (most likely `.exe` or `.msi`) to download.
  2. Execute the Installer
    • Find the downloaded file (usually in the "Downloads" folder).
    • Double-click the file to run it.
  3. Proceed with Installation Instructions
    • Read the setup directions.
    • Accept the license agreement if prompted.
    • Specify the installation directory or use the default.
    • Click "Next" or "Install" to proceed.
  4. Let Installation Complete
    • The installation takes a few minutes.
    • Some applications download additional files.
  5. Open the Software
    • Click on "Finish" if there's an option to open the program.
    • You can also find it in the Start Menu or as a Desktop shortcut.

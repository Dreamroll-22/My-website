Cascading Style Sheets (CSS)

Introduction to CSS

CSS (Cascading Style Sheets) is used to style and layout HTML elements.

Lesson 1: CSS Syntax and Selectors

CSS follows a selector-property-value format.


selector {
    property: value;
}

Explanation:

  • selector → The element you want to style.
  • property → The style attribute (e.g., color, font-size).
  • value → The setting for the property.

Lesson 2: Inline, Internal, and External CSS


/* External CSS */
p {
    color: blue;
}

Explanation:

  • Inline CSS: Applied directly inside an element.
  • Internal CSS: Inside a <style> tag in HTML.
  • External CSS: Stored in a separate .css file.

Lesson 3: Colors and Backgrounds


body {
    background-color: lightblue;
    background-image: url('background.jpg');
    background-repeat: no-repeat;
    background-size: cover;
}

Explanation:

  • background-color → Sets the background color.
  • background-image → Sets a background image.

Lesson 4: Fonts and Text Styling


p {
    font-family: Arial, sans-serif;
    font-size: 18px;
    text-align: center;
}

Explanation:

  • font-family → Changes the font style.
  • text-align → Aligns text left, center, or right.

Lesson 5: CSS Box Model


div {
    width: 200px;
    padding: 10px;
    border: 2px solid black;
    margin: 20px;
}

Explanation:

  • padding → Space inside the border.
  • border → Outline of the element.
  • margin → Space outside the border.

Lesson 6: Positioning and Display


.box {
    position: absolute;
    top: 50px;
    left: 100px;
}

Explanation:

  • position → Specifies how an element is positioned.

Lesson 7: Flexbox for Layouts


.container {
    display: flex;
    justify-content: center;
}

Explanation:

  • display: flex; → Enables flexbox layout.

Lesson 8: Grid Layout


.container {
    display: grid;
    grid-template-columns: 1fr 2fr;
}

Explanation:

  • display: grid; → Enables CSS Grid.

Lesson 9: Media Queries


@media (max-width: 600px) {
    body {
        background-color: yellow;
    }
}

Explanation:

  • @media → Applies styles based on screen size.

Lesson 10: CSS Animations and Transitions


.box {
    transition: background 0.5s ease;
}

.box:hover {
    background: blue;
}

Explanation:

  • transition → Creates a smooth effect.

Final Notes

  • CSS is used for styling web pages.
  • Use Flexbox & Grid for layouts.
  • Use Media Queries for responsive design.

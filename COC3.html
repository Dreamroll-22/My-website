COC3: Setup Computer Server

Set User Account

In Server Computer

  1. Open Server Manager > click Add Roles and features > select Active Directory Domain services (ADDS) Check > Add Features
  2. Features Select DHCP server > check > Add features Select DNS Server > check > Add Features > Click Next > Next > Install > Restart your Computer
  3. Open Server Manager > click Add Roles and features > Click symbol of file and Storage services > click the symbol of file and iscI Services > select file Resource Manager> Check> Add Features Click Next > Next > Install > Restart your Computer

Complete DHCP Configuration

  1. Click Commit > Click close

Promote this Server to a Domain Controller

  1. Select a New Forest > Type the Domain (Ex. tvl.org) > Type NetBIOS domain name will appear automatically > click Next > Next > Install > the Computer will automatically restart

Create Folder in Local Disk D:/E of Server

  1. Computer > Open This PC Locate and open Local Disk D:/E
  2. Right click > Select Folder > type the folder name select folder > Right Click > Properties > click sharing > Click Advanced Sharing
  3. Click Permission Tab>Select Everyone > click Remove > Click Add > Open Advanced > Click Find Now > double click DOMAIN USER > Click ok
  4. Click Caching > check "No file or program from the shared folder are available offline" > click ok > click apply > click ok > copy the Network path > close

Creating Organizational Unit

  1. Open Server Manager > Click Tools Tab > click Active Directory User and Computers (ADUC)
  2. Click Domain > select Domain Controller > Right Click > select New > click Organizational Unit > type name of Organizational Unit (Ex. Ict) > click ok

Creating GPO in Group Policy Management (Folder Redirection)

  1. Open Server Manager > Select tool tab > click Group Policy Management
  2. Click Domain > select Organization unit > Right click > click Create a GPO Open Organizational Unit > select the gpo > right click > click edit
  3. Under User Configuration > select Policy > under Policy click windows setting
  4. Under windows setting click folder Redirection > select Desktop or Document (right click Properties > Under target tab click BASIC > paste the Network Path you copy early > click Apply > click yes > Under setting uncheck Grant > click Apply > click yes

Creating File Screening

  1. Open Server Manager > Select tool tab > click File server Resource Manager > Double click File Screen Management
  2. Double click file screen > Double click Create file Screen > select Browse > Locate the folder in Local Disk D:/E of Server Computer
  3. Click Custom Properties > check Audio or Video files > click OK > select save custom file screen without creating a template > click ok

Creating Domain User

  1. Open Server Manager > Select tool tab > click File server Resource Manager > click Active Directory User Computers > click Domain controller
  2. Click Organization Unit > Right click > click New > Select User > Type the First Name > Type Middle Initial Type The last name > Type the USER LOGON name > Type the password > check Password Never Expired > click Next > click Finish

Creating Domain in Client's Computer

In Client computer

  1. In Desktop > select This PC Right Click > click Properties > Open Change setting > under Computer Name Tab > click Change > Delete the name of workgroup
  2. Check Domain > Type the Domain name > Type the Username and Password > Click ok > Click ok > click restart
  3. Computer will restart > Select Other Account > Type The User log or the logon name > type the Password > enter

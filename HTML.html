Hyper Text Markup Language (HTML)

Introduction to HTML

HTML (HyperText Markup Language) is the foundation of web development. It structures content using tags.

Lesson 1: Basic Structure of an HTML Document

An HTML document follows a standard format:

<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="This is an example webpage">
    <title>My First Webpage</title>
</head>
<body>
    <h1>Welcome to My Website</h1>
    <p>This is my first HTML webpage.</p>
</body>
</html>

Explanation:

  • <!DOCTYPE html> â†’ Declares the document as HTML5.
  • <html lang="en"> â†’ Defines the language of the page.
  • <head> â†’ Contains metadata (title, description, encoding).
  • <meta charset="UTF-8"> â†’ Supports all characters (e.g., emoji ðŸ˜Š).
  • <meta name="viewport" content="width=device-width, initial-scale=1.0"> â†’ Makes the page responsive.
  • <title> â†’ The title seen in the browser tab.
  • <body> â†’ Contains the visible content of the webpage.

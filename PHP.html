Hypertext Preprocessor (PHP)

Introduction to PHP

PHP (Hypertext Preprocessor) is a server-side scripting language used for creating dynamic web pages.

Lesson 1: PHP Syntax & Variables

PHP code is written inside <?php ... ?>. Variables in PHP start with a $ sign.


<?php
    $name = "John";
    $age = 25;
    echo "My name is " . $name;
?>

Lesson 2: Data Types & Operators

PHP supports different data types such as strings, integers, floats, and booleans.


<?php
    $num1 = 10;
    $num2 = 5;
    echo $num1 + $num2; // Output: 15
?>

Lesson 3: PHP Functions

Functions help in reusing code.


<?php
    function greet($name) {
        return "Hello, " . $name . "!";
    }
    echo greet("Alice");
?>

Lesson 4: Conditional Statements (if, else, switch)

Conditional statements help in decision-making.


<?php
    $age = 18;
    if ($age >= 18) {
        echo "You are an adult.";
    } else {
        echo "You are a minor.";
    }
?>

Lesson 5: Loops (for, while, foreach)

Loops allow repeating code multiple times.


<?php
    for ($i = 1; $i <= 5; $i++) {
        echo "Number: " . $i . "<br>";
    }
?>

Lesson 6: PHP Forms & User Input Handling

PHP can handle user input from forms.


<form method="post">
    Name: <input type="text" name="name">
    <input type="submit">
</form>

<?php
    if ($_SERVER["REQUEST_METHOD"] == "POST") {
        $name = $_POST["name"];
        echo "Hello, " . $name;
    }
?>

Lesson 7: PHP Superglobals

Superglobals like $_GET, $_POST, $_SESSION, and $_COOKIE are used to handle data.


<?php
    setcookie("user", "JohnDoe", time() + 3600);
    echo "Cookie Set: " . $_COOKIE["user"];
?>

Lesson 8: PHP File Handling

PHP can read and write files.


<?php
    $file = fopen("sample.txt", "r");
    echo fread($file, filesize("sample.txt"));
    fclose($file);
?>

Lesson 9: PHP & MySQL (Database CRUD Operations)

PHP can connect to MySQL databases.


<?php
    $conn = new mysqli("localhost", "root", "", "test_db");
    if ($conn->connect_error) {
        die("Connection failed: " . $conn->connect_error);
    }
    echo "Connected successfully";
?>

Lesson 10: PHP Sessions & Cookies

Sessions and cookies store user data.


<?php
    session_start();
    $_SESSION["user"] = "Alice";
    echo "Session user: " . $_SESSION["user"];
?>

Lesson 11: PHP Security

Protect PHP applications from SQL injection and XSS attacks.


<?php
    $conn = new mysqli("localhost", "root", "", "test_db");
    $username = mysqli_real_escape_string($conn, $_POST["username"]);
    $sql = "SELECT * FROM users WHERE username='$username'";
?>

Final Notes

  • PHP is widely used for web development.
  • It works with MySQL for database-driven applications.
  • Security practices like input validation are important.

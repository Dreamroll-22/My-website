COC2: Setup Computer Networks

Creating Ethernet Cable

Steps to Create a Straight-Through Cable

  1. Prepare Tools and Materials:
    • Twisted-pair cable (CAT5e or CAT6)
    • RJ45 connectors
    • Crimping tool
    • Cable stripper or cutter
    • LAN tester
  2. Measure and Cut the Cable – Cut the required length of twisted-pair cable using a cable cutter.
  3. Strip the Outer Jacket – Remove about 1 inch of the cable’s outer sheath to expose the twisted wire pairs.
  4. Untwist and Arrange the Wires – Carefully untwist the wire pairs and arrange them based on the wiring standard:
    • T568A Standard
    • T568B Standard
    Ensure both ends follow the same standard for a straight-through cable.
  5. Trim and Insert Wires into RJ45 Connector – Trim the wires evenly and insert them into the RJ45 connector in the correct order. Make sure each wire reaches the end of the connector.
  6. Crimp the Connector – Use the crimping tool to firmly secure the RJ45 connector to the cable.
  7. Repeat for the Other End – Follow the same steps to prepare and crimp the second end of the cable.
  8. Test the Cable – Use a LAN tester to check for proper wiring and connectivity. Ensure all connections are correctly aligned and functional.

Network Configuration

Both client and server computers must do this. The server must do it first before the client.

Change Computer Name

  1. Right-click THIS PC > click Properties.
  2. Click Change Setting > click Change.
  3. Click Computer name > type the Computer name (Ex. Computer1 or Server1).
  4. Click WORKGROUP > type the WORKGROUP (Ex. WORKGROUP) > click OK > click OK > click Restart.

IP Configuration

  1. Click Control Panel > Select Network and Internet.
  2. Click Network and Sharing Center.
  3. Click Change Adapter Settings.
  4. Select Ethernet or Local Area Connection > Right-click Properties.
  5. Click Internet Protocol Version 4 (TCP/IPv4).
  6. Use the following IP Address:
    • IP Address: (Ex. 192.168.1.2)
    • Subnet Mask: (Ex. 255.255.255.0)
    • Default Gateway: Type the Default Gateway of your router (Ex. 192.168.1.1)
    • Preferred DNS Server: (Ex. 192.168.1.1)
  7. Click OK > click OK.

⚠️ Warning ⚠️

The IP address of your server and client must be different from each other.

Turning Off Windows Firewall

  1. Click Control Panel > click System and Security.
  2. Click Windows Defender Firewall.
  3. Click Turn Windows Defender Firewall on or off.
  4. Select Turn Windows Defender Firewall OFF > click OK.

Turning On Network Discovery and Turning Off Password Protected Sharing

  1. Click Control Panel > click Network and Internet.
  2. Click Network and Sharing Center.
  3. Click Change Advanced Sharing Settings.
  4. Click Turn on Network Discovery.
  5. Click Turn off Password Protected Sharing > click Save Changes.

Router Configuration

  1. In the Desktop, right-click Network and Sharing Center > click Properties.
  2. Click Change Adapter Settings.
  3. Select Ethernet > right-click Properties.
  4. Click Internet Protocol Version 4 (TCP/IPv4).
  5. Click Obtain an IP address automatically > click OK > click OK.
  6. Double-click Ethernet > check IPv4 Default Gateway.
  7. Open Google Chrome or Internet Explorer > type the Default Gateway in the address bar.
  8. Enter the router Username and Password (if unknown, check the back of the router).
  9. Click Wireless > go to Basic Settings > click Wireless Network Name and type your preferred network name (Ex. Computer_Router1) > click Save.
  10. Click Wireless Security > click Wireless Password and type your preferred password (Ex. Admin123) > click Save.

WiFi Access

  1. In the Desktop, right-click Network and Sharing Center > click Properties.
  2. Click Change Adapter Settings.
  3. Select Ethernet > right-click Properties.
  4. Click Internet Protocol Version 4 (TCP/IPv4).
  5. Click Obtain an IP address automatically > click OK > click OK.
  6. Open Google Chrome or Internet Explorer.
  7. Type what you want to search (Ex. What is Computer?).

Wireless Printing

Required:

  • USB Dongle
  • Installer for dongle (CD or Flash Drive)
  1. Plug the USB dongle and Installer into the computer.
  2. Click This PC > open the installer (CD or Flash Drive) > install the software.
  3. On the Desktop, right-click Network and Sharing Center > click Properties.
  4. Click Change Adapter Settings.
  5. Right-click Wi-Fi > click Connect.
  6. Select the Wi-Fi network you want to connect to (Ex. Router 1) > click Connect > type the password.
  7. Double-click Network and Sharing Center > find the computer name connected to the printer (Ex. Admin1) > double-click the computer name.
  8. Double-click the Printer Installer on the networked computer > install the printer driver.
  9. To print a file:
    • Select the file (docs, pdf, jpg, etc.) you want to print.
    • Right-click the file > click Print > click Print.
